# Student Record Manager

A Python-based Student Record Management System built using dictionaries, functions and loops.

## Features

- Add Student
- Search Student
- Update Marks
- Delete Student
- View All Students
- Calculate Average Marks
- Generate Grades
- Load Sample Data

## Concepts Used

- Python
- Dictionaries
- Functions
- Loops
- Input Validation

## How To Run

```bash
python student_record_manager.py
```
